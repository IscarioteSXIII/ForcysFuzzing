# Main GUI - Regroupement des différents bout de code.

Features intégrées:
- Interface Graphique (Enzo P)
- Network Scanner (Enzo P)
- Crawler HTTP(s) (Enzo P)
- DDOS (Julien D)

Dépendances :
- NMAP 7.92 - /!\ La version 7.93 n'est pas compatible
- Librairie "requests"
- Librairie "bs4"
- Librairie "scapy"

```
python -m pip install [librairie]
```
